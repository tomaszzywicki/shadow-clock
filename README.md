## Installation 
### 1. Install pipx if you don't have it already:
   - On Arch Linux: 
```bash
sudo pacman -S pipx
```
- On Debian/Ubuntu:
```bash
sudo apt install pipx
```
- On macOS (with Homebrew):
```bash
brew install pipx
```
- On Windows:
```powershell
python -m pip install --user pipx
python -m pipx ensurepath
```

### 2. Clone the repository and install Shadow Clock:
```bash
git clone https://github.com/yourusername/shadow_clock.git
cd shadow_clock
pipx install .
```

### 3. Run Shadow Clock:
```bash
shadowclock
```
